RF1.Apertura de cuenta
El sistema debe permitir crear una cuenta bancaria que pueda iniciar con un saldo de $0 o con un saldo inicial ingresado por el usuario al momento de la creación.

RF2.Depósito de dinero
El sistema debe permitir al usuario realizar depósitos para incrementar el saldo de la cuenta, siempre que el monto ingresado sea mayor a cero.

RF3.Retiro de dinero
El sistema debe permitir al usuario retirar dinero de la cuenta, validando que:
-El monto ingresado sea mayor a cero.
-El saldo disponible sea suficiente para completar la operación.

RF4.Consulta de saldo
El sistema debe permitir al usuario consultar en cualquier momento el saldo actual disponible en la cuenta.

RF5.Menú de operaciones
El sistema debe mostrar un menú interactivo que permita seleccionar las operaciones disponibles:
-Depositar dinero
-Retirar dinero
-Consultar saldo
-Salir del sistema

RF6.Control de ejecución del sistema
El sistema debe continuar ejecutándose hasta que el usuario seleccione la opción de salir del programa.
